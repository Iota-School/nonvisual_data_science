# Nem vizuális adatfeltárás és reprezentáció

Ezen a workshopon megismerkedünk az egyes adattípusokkal való munka és azok reprezentációjának általános módjaival. Ezek közé tartoznak a kategorikus adatok, a bináris adatok, a numerikus adatok és az időadatok. Megfontoljuk, hogy az egyes technikák hogyan helyettesíthetik részben vagy egészben az adattudományok általános vizualizációit, például az oszlopdiagramokat, a szórás diagramokat és a vonaldiagramokat. Azt is megvizsgáljuk, hogy milyen előnyei és hátrányai vannak annak, ha az adatokat a  Python nyelvvel beszélgetve tárjuk fel, ahelyett, hogy egyedi meghangosításokat vagy vizualizációkat hoznánk létre.
