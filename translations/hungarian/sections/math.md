# további matematika

Oké, megtanultuk az áttekintés alapjait. Próbáljunk ki még egy kis matematikát. Írjuk be ezt a négy sort a REPL-be:

```python
3 + 3
```

Ez egy hármas, egy pluszjel és egy hármas.

``python
10 - 4
```
Ez egy tízes, egy mínuszjel (vagy kötõjel) és egy négyes.

```python
5 * 5
```

Ez egy ötös, egy csillag és egy ötös.

```python
9 / 3
```

Ez egy kilences, egy perjel és egy hármas.

A Pythonban a kivonásra mínusz (`-`)jelet (vagy kötõjelet), a szorzásra csillagot vagy csillagot (`*`), az osztásra pedig egy perjelet(`/`) használunk. Vannak más matematikai operátorok is, de most maradunk ezeknél.
