# Telepítés

Ez a workshop a Windows platformra összpontosít, mivel részletesen foglalkozik a Windows-specifikus NVDA képernyõolvasó használatával.

## Az NVDA telepítése

Az NVDA, vagyis a Non-Visual Desktop Access egy képernyõolvasó a Windows operációs rendszerhez. Az NVDA telepítéséhez egy Windows számítógépre látogasson el az [NVDA letöltési oldalára](https://www.nvaccess.org/download/), és töltse le a szoftvert. 
(Ha más képernyõolvasót használ, a letöltési link az elsõ gomb az oldalon). Futtassa a telepítõ programot, és telepítse az alapértelmezett beállításokkal.

Az NVDA telepítés utáni indításához nyomja meg a Windows gombot a billentyûzetén (más néven meta gomb, leggyakrabban a Control és az Alt gomb között található a billentyûzet bal alsó részén). A képernyõolvasó elindításához írja be az nvda szót, majd nyomja meg az Entert.

## Az Anaconda telepítése

Az Anaconda egy Python-disztribúció, vagyis egy olyan szoftvercsomag, amely a Python mellett számos más hasznos eszközt és segédprogramot is tartalmaz. Az Anaconda kifejezetten az adattudomány támogatására készült...

Az Anaconda telepítéséhez kövesse az alábbi lépéseket:

1. Látogasson el az [Anaconda letöltési oldalára](https://www.anaconda.com/download/).
2. Kattintson a Letöltés linkre. (Ez a második link a "Free Download" címszó után.)
3. Futtassa a telepítõ programot.
4. Telepítse az alapértelmezett beállításokkal.

Ebben a workshopban az Anaconda Promptot fogjuk használni (nem az Anaconda Navigátort). 
Az Anaconda prompt megnyitásához nyomja meg a Windows gombot a billentyûzeten (más néven a meta gombot, általában a billentyûzeten a  Control és Alt gombok között található.). 
Írja be az "anaconda prompt" szót, és az Anaconda Promptnak kell lennie az elsõ találatnak. Nyomja meg az Enter billentyût a program elindításához.
